Foundation prerequisites

Token sets that must land before specific Guidelines sections become useful.

  • Motion / animation tokens Proposed — durations, easings, transition recipes. Unblocks the Motion section in component Guidelines.
  • Breakpoint tokens Proposed — responsive breakpoint set. Unblocks the Responsive Behavior section in component Guidelines.

Site features

  • Token search Shipped — global header search box and keyboard shortcut (/ or ⌘K) across all UDS tokens. Click a result to copy var(--name).
  • Inline spec editing Proposed — designers edit Guidelines via a schema-generated form, saves create a GitHub PR.
  • Figma → JSON sync Shipped — lands as a set of agent-facing Cursor skills (import-figma-tokens, sync-figma-component-spec, link-figma-nodes) rather than the originally-proposed server-side action. A designer types “UDS updated” (or the per-task variant) and the right skill round-trips Figma into uds/tokens/*.css or the relevant uds/components/<id>/spec.json.
